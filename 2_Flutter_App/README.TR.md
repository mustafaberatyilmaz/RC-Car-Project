# 2. Flutter Mobil Uygulama (Flutter App)

Bu klasör, RC aracı kontrol etmek için geliştirilen mobil uygulamanın kaynak kodlarını içerir.

## İçerik
- **flutter_app:** Mobil uygulama kaynak kodu.
- **screenshots:** Uygulama arayüz görselleri.

## Özellikler
- Araç kontrol arayüzü.
- Wi-Fi üzerinden ESP32 bağlantısı.
- Video akışı görüntüleme.
