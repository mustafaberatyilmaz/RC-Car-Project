# 2. Flutter Mobile App

This directory contains the source code for the mobile application developed to control the RC car.

## Contents
- **flutter_app:** Mobile application source code.
- **screenshots:** Application interface images.

## Features
- Vehicle control interface.
- ESP32 connection via Wi-Fi.
- Video stream viewing.
